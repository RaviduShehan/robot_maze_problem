# robot_maze_problem
Robot-maze problem using B langauge

Developed a Robot moving around a simple maze using B tools Atelier B & ProB
Maze is a grid of squares 7 wide by 5 high
Starting point is (1,1)
Robot can move in the maze "Path" but shoudl avoid the internal walls in the maze
Robot can teleport inside the maze in its path. But he can not immedietly teleport to exist
